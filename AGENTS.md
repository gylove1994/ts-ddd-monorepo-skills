# Agent Instructions

This repository is a multi-agent skills template.

Before starting a task that may use repository-specific workflows:

1. Inspect `skills/` for a relevant skill.
2. Read that skill's `SKILL.md`.
3. Follow the skill instructions exactly when they apply.
4. Prefer agent-neutral skill content over platform-specific copies.

Skills are intentionally stored once in `skills/`. Platform entrypoints exist only to help different agents discover the same library.
