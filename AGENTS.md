# Agent Instructions

This repository is a multi-agent skills template.

Before starting a task that may use repository-specific workflows:

1. Inspect `skills/` and `.agents/skills/` for relevant skills.
2. Read that skill's `SKILL.md`.
3. Follow the skill instructions exactly when they apply.
4. Prefer agent-neutral skill content over platform-specific copies.

Distributed skills are intentionally stored once in `skills/`. Project-level authoring, testing, or repository operation skills may live in `.agents/skills/`. Platform entrypoints exist only to help different agents discover the same library.
